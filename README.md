I built a BookMyShow-style application that lets people easily browse movies, pick seats, and book tickets in a simple and smooth way. I created it to understand how real-world ticketing systems work and to practice building something people would actually use.
I used Spring Boot because it helps me build backend services quickly with a clean structure and less setup.
I used MySQL because it provides reliable storage for structured data like users, movies, and bookings.
I used Redis because it improves performance by caching frequently accessed data and reducing database load.
I used JWT because it enables secure user authentication without maintaining server-side sessions.
I used Docker because it makes the application easy to run and deploy consistently across different environments.
